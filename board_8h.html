typedef unsigned long Board[2]

Plateau de jeu.

Un plateau de jeu est un tableau de 2 entiers non signés.

Le premier entier représente les pions noirs. Le second entier représente les pions blans.

Chaque bit représente une cellule du plateau.

0 = Cellule vide 1 = Cellule pleine
